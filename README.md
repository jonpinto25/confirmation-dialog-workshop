# React Confirmation Dialog Workshop

This repository contains a **hands-on workshop project** for building a reusable confirmation dialog in React using **Context** and **JavaScript Promises**.

The goal of the workshop is to show how to:

- Use React Context to provide a global dialog component.
- Handle user confirmation using async/await with Promises.
- Keep business logic separated from UI.
- Build reusable and composable components in React.

### Install dependencies

`npm install`

### Install node 22

`nvm install 22` 

`nvm use`

### Run the project

`npm run dev`

### See the project

Open http://localhost:5173
